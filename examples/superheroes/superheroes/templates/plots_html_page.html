Superheroes - Statistics

This page shows general statitics about the ozelot example project 'superheroes'.

The characters analyzed here are limited to the following universes (see also the example's documentation): {{ universes_list }}.

The analyses shown here are not performed with rigorous statistical methods and ignore, for example, issues in analyzing small numbers of samples. This page serves only to demonstrate data visualization and output generation.

Number of characters and movies

The following two plots show the number of first character appearances per year and the number of published movies per year.

{{ first_appearances_per_year_svg }}
{{ movies_per_year_svg }}

Both counts grow strongly over the years, with the number of first appearances growing faster than the number of movies. Note that a 'first appearance' doesn't have to be a movie appearance but may also be in some other form of media.

Let's look at the average number of character appearances in a movie (all, not only first appearances) per year.

{{ average_appearances_per_movie_svg }}

While the number of first appearances of characters grows substantially over the years, the total number of characters per movie stays relatively constant between 1995 and 2016.
